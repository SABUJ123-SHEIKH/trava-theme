There are 6 blocks in total in trava-theme-guaranteed, rules for use:
1. Trava Header
* Search for “Trava Header” in the Block editor
* Set logo, menu item, CTA button
2. Trava Hero
* Enter “Trava Hero”
* Set heading, subheading, CTA text/link, image
3. Trava Services
* Enter “Trava Services”
* Add/remove service card, set icon/title/text
4. Trava Testimonials
* Enter “Trava Testimonials”
* Add/remove testimonial list, set slider settings
5. Trava CTA
* Enter “Trava CTA”
* Enter heading, text, button/link, background image
6. Trava Footer
* Enter “Trava Footer”
* Set nav/service link, contact, social link
Tip: To find blocks in the editor Just type "Trava" and everything will show up.
